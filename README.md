# 🧪 Pruebas unitarias y de integración con JUnit 5, Mockito y Spring Boot

> Implementación de tests unitarios, de servicios, repositorios y controladores REST
> aplicando TDD y las mejores prácticas de testing en Java.

---

## 📖 Descripción

Proyecto de pruebas en Java que cubre desde tests unitarios con JUnit 5 y Mockito
hasta pruebas de integración completas con Spring Boot, aplicando las mejores
prácticas de TDD y testing por capas: servicios, repositorios y controladores REST.

---

## 🗂️ Lo que he practicado

| # | Bloque |
|---|--------|
| 1 | JUnit 5 |
| 2 | Mockito |
| 3 | Spring Boot: Test de Servicios con Mockito |
| 4 | Spring Boot: Test de Repositorios con DataJpaTest |
| 5 | Spring Boot: Test de Controladores con MockMvc (WebMvcTest) |
| 6 | Spring Boot: Test de integración de Servicios REST con WebTestClient |
| 7 | Spring Boot: Test de integración de Servicios REST con TestRestTemplate |

---

## 🎯 Habilidades adquiridas

- Escribir pruebas unitarias con **JUnit 5** desde cero
- Usar **Mockito** para simular dependencias y verificar comportamiento
- Probar la capa de servicios de Spring Boot con **Mockito**
- Probar repositorios JPA con **@DataJpaTest**
- Probar controladores REST con **MockMvc** y **@WebMvcTest**
- Realizar pruebas de integración con **WebTestClient** y **TestRestTemplate**
- Aplicar las mejores prácticas de **TDD** en proyectos reales
